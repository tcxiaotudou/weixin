

# weixin
> 仿微信即时通讯WebApp

#### 技术栈：

> 前端：MUI框架
>
> 后端：SpringBoot + Mybatis + Netty + FastDFS

#### 主要功能

1. 用户注册/登录
2. 本地聊天记录，快照等数据缓存
3. 联系人列表
4. 添加好友，支持用户名搜索和二维码扫一扫
5. 消息音效提示和自动重连
6. 个人信息修改
